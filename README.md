# Video Dataset - 1,300+ files

The dataset comprises **1,300+** videos of **300+** people captured using mobile phones (including Android devices and iPhone) and webcams under varying lighting conditions. It is designed for research in **face detection**, **object recognition**, and **event detection**, leveraging high-quality videos from smartphone cameras and webcam streams.  ‚Äî **[Get the data](https://unidata.pro/datasets/phone-and-webcam-video/?utm_source=github-biometrics&utm_medium=referral&utm_campaign=phone-and-webcam-dataset)**
